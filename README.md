# AMK

Данный проект посвящён разработке кроссплатформенного мобильного приложения для сотрудников организации "AMK-групп".  
Приложение предназначено для централизованного доступа к корпоративным новостям, внутренней информации компании и отраслевым новостям автомобильного бизнеса.

## Цель

Создание удобного и функционального мобильного приложения, которое:
- объединяет корпоративные сервисы;
- обеспечивает быстрое распространение внутренней информации;
- формирует единое информационное пространство для сотрудников компании.

## Роли и привилигии

 Пользователь - сотрудник компании, может просматривать контент
 Адинистратор - сотрудник компании имеющий полный доступ к управлению системой и пользователями.
